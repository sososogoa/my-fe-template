# My Frontend Template

A modern, minimal starter template for frontend projects using:

- **React 19**
- **Vite 6**
- **Tailwind CSS 4**
- **TypeScript**
- **ESLint (flat config) + Prettier**
- **Alias (@/...) for clean imports**
- (Optional) Testing ready with Vitest (to be added)

---

## Getting Started

```bash
npm install
npm run dev
```
